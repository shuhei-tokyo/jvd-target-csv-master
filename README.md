# jvd-target-csv-master
JRA-VAN DataLab. Target にて出力する csv ファイルを用いて、各種マスターテーブルを作成するための issue 中心のレポジトリ

## Target 開催成績 csv 出力項目

- 赤マーカ: 対応する jvd コードあり
- 青マーカ: 対応する target コードあり
